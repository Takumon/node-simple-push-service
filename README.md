# node-simple-push-service
## 概要
[simple-push-service](https://github.com/uemu/simple-push-service)のNode 版

## 前提条件
Nodeの環境

## 実行方法
* クローンして...
```
$ git clone https://github.com/Takumon/node-simple-push-service.git
```
* プロジェクトルートに移動して...
```
$ cd node-simple-push-service
```
* サーバ起動!!
```
$ npm start
```
